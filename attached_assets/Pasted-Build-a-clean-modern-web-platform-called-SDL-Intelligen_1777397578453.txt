Build a clean, modern web platform called "SDL Intelligence" — a global knowledge hub for Self-Directed Learning (SDL) research, powered by the RECAST team at Florida State University (led by Dr. Bret Staudt Willet) and the SDL Research Lab at North-West University (NWU), South Africa.

TECH STACK
- React (Vite)
- Plain CSS (no Tailwind, no component libraries)

- No backend yet — all AI calls will be added later via Claude API and Gemini API

Do NOT include any Replit-specific configuration, branding, or dependencies. The output must be generic and portable.

DESIGN INSPIRATION
Model the hero section on the style of https://arcvale.com:
- Full-viewport dark hero section
- Bold, all-caps large headline centered on screen
- Cinematic dark background — use a deep navy-to-black gradient with a subtle animated particle or floating-node effect (CSS/JS only, no canvas libraries)
- Subheading in a lighter weight, muted white/gray tone
- A single prominent CTA button with a hover glow effect
- Minimal top navigation bar: logo text on the left, nav links on the right (transparent background that becomes solid on scroll)
- No stock photos, no hero images — pure CSS/gradient atmosphere
